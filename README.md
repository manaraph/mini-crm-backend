# mini-crm-backend

## Cloning Repository
> git clone git@github.com:manaraph/mini-crm-backend.git

## Installation and running app with npm
``` bash

cd mini-crm-backend

# install dependencies
npm install 

# serve app at localhost:4000
npm start

```
## Installation and running app with yarn
``` bash

cd mini-crm-backend

# install dependencies
yarn 

# serve app at localhost:4000
yarn start

```
